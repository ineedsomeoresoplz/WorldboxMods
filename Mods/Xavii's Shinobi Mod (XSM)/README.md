# Xavii's Shinobi Mod (XSM)

Naruto mega-pack with clans, dojutsu, chakra, weapons, and new ascension paths.

### Features
- Clan lines: Uchiha, Hyuga, Senju, Uzumaki with dojutsu awakenings (Sharingan chain to Mangekyo/Eternal/Rinnegan, Byakugan to Pure).
- Rank ladder: Academy Student -> Genin -> Chunin -> Jonin -> Anbu -> Anbu Captain -> Kage -> Former-Kage with kill/level thresholds.
- Ascension tier: Sage Mode, Baryon Mode, and Six Paths unlockable via Uzumaki/Senju kill streaks and chakra nature mastery (toggle in config).
- Chakra styles: five natures with unique stat curves plus item set (Samehada, Executioner Blade, Uchiha Fan, Kusanagi).
- Legendary shinobi: Hashirama, Itachi, Obito, Madara, Final Form plus revamped bijuu and legacy trait groups.
- Status effects and VFX: Amaterasu, Kamui, Susano forms, genjutsu, woodstyle roots, lightning, meteors.

### Installation
1. Drop Xavii's Shinobi Mod (XSM) into WorldBox/Mods.
2. Load/build with NeoModLoader.
3. Adjust default_config.json or in-game mod options for clan names, auto-favorite, force rename, unlock legends, and Legendary Ascension.

### Compatibility
- Built for WorldBox build 719 with NeoModLoader.
- Legacy trait groups remain for older saves; new ascension path is opt-in.
